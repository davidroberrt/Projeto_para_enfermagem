# Projeto_para_enfermagem
Esse jogo foi feito para revisão das provas de Bioagentes Patogênicos e apresentado na sala de aula para os alunos do 2º período do curso de enfermagem da faculdade IESP. Matéria: Bioagentes patogênicos. O jogo foi feito na linguagem python. você precisará baixar as bibliotecas: tkinter e pygame.
